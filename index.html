Multi Almacenes Bolivia

Sistema gratuito para la gestión de almacenes.

Características Generales

  • Información del cliente personalizable
    • logotipo
    • información de la empresa
  • Parametros de almacén :
    • 10 Almacenes (en el sistema).
    • 7 usuarios (en el sistema).
    • logs / registros del sistema (diariamente).
    • articulos (por almacen).
    • 500 proveedores (en el sistema).
  • Colores según cantidad en stock :
    • ROJO (sin stock.)
    • AMARILLO (stock mínimo.)
    • VERDE (stock suficiente.)
  • Tipos de documentos utilizados en los movimientos de articulos.
  • Parametros del sistema configurable
    (archivos de configuración cliente y servidor) en formatos : YML , JS , JSON.
  • Base de Datos Sqlite (sin configuración y portable).
  • Frontend VueJs (capa de usuario - vista).
  • backend .Net Core (Interfaz de programación de aplicaciones - API).
  • Autenticación basada en Tokens (Json Web Tokens - JWT).
  • Usuarios :
    • 1 Usuario por defecto (Administrador).
    • 6 Usuarios gestionables por un usuario administrador.
    • Tipos de usuarios (Administrador y Operador)
    • Contraseñas modificables pero no recordables.
  • Tablas de resultados :
    • Gráficas.
    • Comparación mensual.
  • Reportes :
    • PDF formato de impresión.
  • Exportación de datos XLSX - Excel.
  • Metodos de inventario :
    • PEPS (Primero en entrar Primero en salir).
    • UEPS (Último en entrar Primero en salir).
    • PP (Precio Promedio).
  • Consultas :
    • Por proveedor.
    • Por articulo.
    • Por almacén.
  • Logs de acciones realizadas (registros del sistema).
  • Idiomas :
    • Español.
    • Ingles.
  • Temas :
    • Iluminado.
    • Oscuro.

Requisitos

Para utilizar el aplicativo su equipo deberá cumplir como mínimo las siguientes características

  • Sistema Operativo
    • Windows 7 , 8 , 10 (64bits)
    • Debian 9, Ubuntu 17, Elementary Loki, Mint 17 (64bits)

  • Navegador Web
    • Google Chrome v25+
    • Opera v25+
    • Brave v25+

  • Pantalla
    • Responsive / Adaptable. (Escritorio, Portatil, Tableta , Televisor.)
    • Resolución minima recomendada 1024 x 768.

  • Recursos
    • RAM 512Mb
    • HDD 80Gb
    • Procesador Pentium IV (2.4GHz)
