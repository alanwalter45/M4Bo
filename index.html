Multi Almacenes Bolivia (M4Bo)

Sistema de gestión de almacenes.

Características Generales

  • Información del cliente personalizable
    • logotipo
    • información de la empresa
  • Gestión de almacén :
    • 10 Almacenes (en el sistema).
    • 7 usuarios (en el sistema).
    • logs / registros del sistema (diariamente).
    • articulos (por almacen).
    • 300 proveedores (en el sistema).
  • Alerta por colores según cantidad en stock:
    • ROJO sin stock.
    • AMARILLO tiene el stock mínimo.
    • VERDE tiene stock suficiente.
    • Notificaciones via email según color (ROJO,AMARILLO,VERDE).
  • Paleta de 25 Articulos favoritos.
  • Código de barras para los articulos.
  • Ubicación de los articulos rack / secciones de ubicación.
  • Resguardo de articulos ajenos documentos de pruebas.
  • Separar articulos por lotes agrupación de articulos.
  • Archivos de autenticidad documentos digitales de la adquisisión de articulos.
  • Recordar completar entrega cuando se realizan entregas por parte de proveedores pero no están completas o están dañadas y se necesita reponer.
  • Recordar completar reposiciones cuando se realizan entregas a clientes pero no están completas y se necesita reponer.
  • Control de acceso (seguridad) al almacén planilla de asistencia.
  • Parametros del sistema configurable archivos de configuración servidor y cliente.
    Formatos : YML , JS , JSON
  • Base de Datos Sqlite (sin configuración y portable).
  • Frontend VueJs (vista/ capa de usuario).
  • Autenticación basada en Tokens (JWT).
  • Operadores / Usuarios:
    • 1 Usuario por defecto (Administrador).
    • 6 Usuarios gestionables por un usuario administrador.
    • Tipos de usuarios Administrador y Operador
    • Contraseñas modificables pero no recordables (Hash Key).
  • Tablas de resultados:
    • Gráficas circulares.
    • Comparación mensual.
  • Reportes :
    • PDF formato de impresión.
  • Exportación de datos XLSX - Excel.
  • Metodos de inventario:
    • PEPS PRIMERO EN ENTRAR PRIMERO EN SALIR FIFO
    • UEPS ULTIMO EN ENTRAR PRIMERO EN SALIR LIFO
    • PP PRECIO PROMEDIO PRO o PONDERADO
  • Consultas en linea ( 1 mes ) :
    • Por proveedor.
    • Por articulo.
    • Por almacén.
  • Respaldo y restauración de información (Backups).
  • Logs de acciones realizadas (registros del sistema).
  • Idiomas soportados
    • Español.
    • Ingles.
    • Portugues.
  • Temas (Colores, Fuentes , Paneles...)
    • Light / Iluminado.
    • Dark / Oscuro.

Requisitos

Para que pueda utilizar el aplicativo su equipo deberá cumplir como mínimo estas características

  • Sistemas Operativos
    • Windows 7 , 8 , 10
    • Debian 9, Ubuntu 17, Elementary Loki, Mint 17
    Para arquitecturas de 64 bit.

  • Navegadores Web
    • Opera
    • Google Chrome
    Para versiones superiores a 25

  • Pantalla
    • Responsive / Adaptable. (Escritorio, Portatil, Tableta , Televisor.)
    • Resolución minima recomendada 1024 x 768.
